### 项目简介

本项目的目的是学习并实现Q-Learning及DQN算法，算法基于的游戏环境是一个基于Pygame开发的FlappyBird游戏。

运行Game.py可以运行游戏。

运行Q-learning.ipynb可以运行Q-learning算法。

运行DQN.ipynb可以运行DQN算法。

另外，本项目已实现全局优化算法的接口，并实现了一个群体模拟退火算法的简单例子，详见SA.ipynb.


### 安装本项目依赖


```sh
pip install -r requirements.txt
```
